🎬 PlayZilla – Mobile Cinema Player

PlayZilla is a simple and modern web based media player designed with a clean cinematic UI.  
This project was created for fun and learning purposes, inspired by mobile streaming apps, with a strong focus on **mobile responsiveness** and minimal design.

🌐 Live Demo
👉 https://god-stream.vercel.app

✨ Features
- 🎥 Clean & modern media player UI
- 📱 Fully responsive (mobile-first design)
- ⚡ Smooth user interaction
- 🎨 Dark cinematic theme
- 📂 Local movie file selection
- 🚀 Fast loading & lightweight

 🛠️ Built With
- HTML5 – Structure  
- CSS3 – Styling & animations  
- JavaScript – Media control logic  

📱 Mobile Support
PlayZilla is optimized for mobile devices, providing a smooth and user-friendly experience on:
- Android phones
- iOS devices
- Tablets
- Desktop browsers

📂 Project Purpose
This project was built:
- For fun & UI experimentation
- To practice front-end development skills
- As a showcase project for GitHub profile

🚀 Deployment
The project is hosted using Vercel for fast and reliable performance.
